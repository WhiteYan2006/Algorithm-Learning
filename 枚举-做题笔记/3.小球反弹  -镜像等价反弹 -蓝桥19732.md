# 问题描述

有一长方形，长为 $343720$ 单位长度，宽为 $233333$ 单位长度。在其内部左上角顶点有一小球 (无视其体积)，其初速度如图所示且保持运动速率不变，分解到长宽两个方向上的速率之比为 $dx:dy=15:17$。小球碰到长方形的边框时会发生反弹，每次反弹的入射角与反射角相等，因此小球会改变方向且保持速率不变（如果小球刚好射向角落，则按入射方向原路返回）。从小球出发到其第一次回到左上角顶点这段时间里，小球运动的路程为多少单位长度？答案四舍五入保留两位小数。

# 实现
## 思路
- 对于反弹问题，可以理解为镜像，令长方形翻转复制，碰到底部则向下翻转，碰到右侧则向右翻转

![[Drawing 2025-07-17 11.09.00.excalidraw]]
- 任意坐标为$\left( 2nx,2my \right)$的点都是原点的镜像
- 枚举$t$，看$\left( 15t,17t \right)$能否分别被$\left( 2x,2y \right)$整除

## 代码
```C++
#include <bits/stdc++.h>
using namespace std;
int main()
{
  // 请在此输入您的代码
  long long dx=15,dy=17;
  long long x=343720,y=233333;
  for (int t=1; ; t++) {
    if (t*dx % (2*x)==0 && t*dy%(2*y)==0) {
      cout.precision(2);
      cout << fixed;
      cout << hypot(dx*t,dy*t); // 很方便的计算勾股定理的函数
      break;
    }
  }
  return 0;
}
```
- `hypot()`函数可以快速求解勾股定理
